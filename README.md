# 🚢 Titanic Data Cleaning & Visualization

A beginner-friendly Data Cleaning and Visualization project using the Titanic dataset with Python.

## 📌 Project Overview
This project demonstrates the complete data preprocessing workflow:
- Load the Titanic dataset
- Handle missing values
- Remove unnecessary columns
- Perform Exploratory Data Analysis (EDA)
- Visualize key insights using charts

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📂 Project Workflow
1. Import Libraries
2. Load Titanic Dataset
3. Explore Dataset
4. Handle Missing Values
5. Drop Unnecessary Columns
6. Visualize Data
7. Generate Correlation Heatmap

## 📊 Visualizations
- Survival Count
- Survival by Gender
- Survival by Passenger Class
- Age Distribution
- Correlation Heatmap

## 📈 Key Insights
- Most passengers did not survive.
- Female passengers had a higher survival rate.
- First-class passengers survived more often.
- Age influenced survival.
- Higher fare passengers generally had better survival chances.

## ▶️ Run the Project

```bash
pip install pandas numpy matplotlib seaborn
python data_cleaning.py
```
